READ ME

Note: no third party libraries or modules. All graphics done in tkinter.

This is a game nameed "bloxorz&solver&design".

In the game, the player can use arrow keys to control the block, to flip it around.
The goal is to let the block flip into the hole.
There are many features in the game, such as switches and dividers.

For the boards without any swithes or dividers, the player can press h for hint and q for best solution.
If one presses h, the next step will turn green.
If one presses q, the computer will flip the block by itself.

In the design board section, press r to go back to home page.
Press space to save the newly created board into load levels.
Press c to clear the board.
